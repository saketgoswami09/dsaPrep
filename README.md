## 📁 Folder Structure

| Folder | Topics Covered |
|---|---|
| `Arrays/` | Two Sum, Kadane's, Prefix Sum, etc. |
| `Strings/` | Anagram, Palindrome, Sliding Window on strings |
| `LinkedList/` | Reversal, Cycle Detection, Merge |
| `Trees/` | BFS, DFS, BST, LCA, Diameter |
| `Graphs/` | BFS, DFS, Dijkstra, Union-Find, Topological Sort |
| `DynamicProgramming/` | 0/1 Knapsack, LCS, LIS, Memoization |
| `Sorting/` | QuickSort, MergeSort, Counting Sort |
| `BinarySearch/` | Classic BS, BS on Answer |
| `Stacks_Queues/` | Monotonic Stack, Deque, Next Greater Element |
| `Recursion_Backtracking/` | Permutations, Combinations, N-Queens |
| `Heaps/` | Top-K, Median Finder, Priority Queue |
| `Tries/` | Word Search, Autocomplete |
| `TwoPointers/` | Container With Most Water, 3Sum |
| `SlidingWindow/` | Max Subarray, Longest Substring |
| `Math_Bit/` | Bit Manipulation, GCD, Sieve, Power |

## 📝 File Naming Convention

```
<ProblemNumber>_<ProblemName>.<ext>
```

- `1_TwoSum.py`
- `206_ReverseLinkedList.java`
